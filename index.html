Abstract

This project presents an interactive data analysis dashboard built with Python and OpenAIâ€™s GPT-4.1-mini API.
The system automatically processes two Excel files (one containing tree-cutting permits and another containing appeals), cleans and merges the data, extracts insights using GPT-based reasoning, and visualizes key statistics through interactive charts.
The dashboard provides valuable environmental insights and supports transparent, data-driven decision-making for forestry management.

Examples

Editing Appellants File

The original appeals file included unstructured text fields difficult to analyze. After processing with the GPT-4.1-mini API, each row is interpreted and summarized into structured, readable columns.

Editing Trees File

The original trees file contained multiple sheets and inconsistent formatting. After processing, all relevant information was merged into one clean, unified sheet ready for analysis.

These are some of the charts generated by the dashboard - showing city distributions, tree species statistics, and the most common reasons for cutting or preserving trees.

Technical Details

The dashboard was developed using modern data engineering and AI tools to automate environmental Excel reporting.

  • Backend: Python (Flask), Pandas, NumPy, tqdm.
  • AI Integration: OpenAI GPT-4.1-mini API for structured text interpretation.
  • Frontend: HTML, CSS, JavaScript, Chart.js.
  • Visualization: Interactive charts displaying cities, species, reasons, and appeals.
  • Execution: Runs locally using a Python Flask server.
